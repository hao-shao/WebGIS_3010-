------------------------------------------------------------------武汉市沙湖公园地图与导航服务网站说明----------------------------------------------------------
本网站用于宣传和服务武汉市沙湖公园，可添加符合、点线面等也可用于导航和反馈，分为如下几个模块：
（1）Index-首页。
        用于说明沙湖公园概述和展示图片。
（2）Arcgismap-ArcGIS地图展示。
        通过定义三维和二视图，可展示公园的卫星影像和地图，地图联动使操作更加方便；
        地图左下方和右下方分别展示了坐标和比例尺；
        地图右上方可更换白，灰，和黑色主题；
        右侧可加载全国道路数据，还可清除；
        点击“符号绘制”进入绘制页面。
（4）drawsym-地图符号绘制。
        地图右侧Editor可自定义添加和修改地图符号，可添加描述、状态和优先权等属性，用于添加公园内的人群活动区、植被区、公共服务区、禁止活动区等，用于公园规划；
        地图左侧可添加点线面等符号，用于公园的划区规划；
（5）Navigation-百度地图导航。
        通过调用百度地图api，实现路径导航功能，其中输入"城市名称"和"检索名称"可线上展示某一地物（如“武昌区”“公园”）；
        输入“起点”和“终点”可实现路径导航，并且具有字符提示功能（如”湖北大学”“沙湖公园”）；
        地图下方还可分页展示检索名称和规划路线。
（6）contact-反馈功能。
       填写相关信息用于反馈，并可识别所填区域是否为空。
（7）湖北省边界。
